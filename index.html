IoT Botnet

O que é?

Uma botnet de IoT é uma rede de dispositivos conectados à internet das coisas (IoT) , normalmente roteadores, que foram infectados por malware (especificamente malware de botnet de IoT) e caíram sob o controle de agentes mal-intencionados. As botnets de IoT são conhecidas por serem usadas para lançar ataques distribuídos de negação de serviço (DDoS) contra entidades-alvo para interromper suas operações e serviços. Para que serve?

Um botnet de IoT é criado usando malware de IoT. Os criminosos cibernéticos geralmente têm como alvo os dispositivos de IoT ao criar botnets, pois eles geralmente têm pouca segurança. Por exemplo, grandes botnets de IoT foram criados tentando fazer login usando credenciais padrão ou explorando vulnerabilidades não corrigidas no software do dispositivo de IoT. Essas falhas na segurança da IoT permitem que um invasor obtenha acesso ao dispositivo e execute o malware de botnet. Depois que o malware de botnet é instalado em um dispositivo de IoT, o dispositivo pode ser controlado remotamente para cumprir as ordens do invasor. Por exemplo, um bot de IoT pode ser instruído a participar de um ataque DDoS contra um endereço específico.

Como se prevernir?

Para evitar se tornar parte de uma botnet IoT, é melhor tomar algumas precauções simples que comprovadamente têm efeito. Primeiro, sempre troque as senhas dos dispositivos quando você compra produtos ou sistemas novos — muitas botnets pegam automaticamente dispostivos com uma senha padrão. É também crucial manter o firmware atualizado, recentemente os fabricantes costumam atualizar continuamente a segurança dessas funções por meio destas atualizações. Evite deixar expostos seus dispositivos diretamente na Internet (como acessíveis até mesmo câmeras sem senha) e, se possível, desative os serviços que você não usa, como acesso remoto.
